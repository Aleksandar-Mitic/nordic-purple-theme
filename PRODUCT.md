# Product

## Register

product

## Users

Desktop Linux developer maintaining a polished Ubuntu/GNOME workstation used for everyday web development.

## Product Purpose

Nordic-Purple provides a reproducible personal GTK/GNOME Shell theme that preserves Nordic's calm dark base while replacing the default teal/blue accents with a consistent purple accent system.

## Brand Personality

Calm, precise, purple.

## Anti-references

Avoid generic teal Nordic defaults, unreadable low-contrast titlebar controls, and fragile local-only tweaks that cannot be reinstalled after a system rebuild.

## Design Principles

- Preserve Nordic's dark, low-glare desktop feel.
- Make purple the only primary accent family.
- Keep GTK3, GTK4, GNOME Shell, and Cinnamon assets visually aligned.
- Prefer reproducible installer-backed changes over one-off local edits.

## Accessibility & Inclusion

Keep text and titlebar controls legible on dark surfaces. Avoid relying on color alone for critical UI states where the upstream theme already provides shape or icon differences.
